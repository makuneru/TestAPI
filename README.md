#Setup
1. Download and install node.js (LTS)
2. Donwload and install Microsoft Visual Studio Code. 
3. Open VS Code
4. Clone this repository
5. Open new terminal
6. Install all dependencies using -> npm install
7. Run all test using this command -> npm run test
